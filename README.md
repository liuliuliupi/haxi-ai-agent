# 项目名称
AI自由行行程规划系统
# 项目介绍
Haxi AI Agent 是一个基于 Java Spring Boot 和 Vue 3 构建的综合智能 Agent 平台，旨在展示包括多 Agent 模式、检索增强生成（RAG）和模型上下文协议（MCP）集成在内的高级 AI Agent 能力。该平台为构建具有自主推理、工具执行和流式响应功能的 AI 驱动应用程序提供了生产就绪的基础。
该平台遵循现代微服务架构，在 AI 推理、工具执行、数据检索和用户界面层之间实现了清晰的分离。后端利用 Spring Boot 3.5.11 和 Java 21，集成了 Spring AI Alibaba Agent Framework 用于核心 Agent 编排；而前端通过 Vue 3 提供响应式体验，并支持实时流式传输。
